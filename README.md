# Wander Mode – JS/XML Final APIs Project

Wander Mode is a small web app that lets you:

- Teleport to a random city (London, Paris, New York, Tokyo, etc.)
- View the street using **Google Street View**
- See **nearby cafés** using **Google Places**
- Click a café to teleport the Street View to that location

It’s built with:

- **Client:** React + Vite
- **Server:** Node.js + Express
- **APIs:** Google Maps Embed (Street View) + Google Places Nearby Search
